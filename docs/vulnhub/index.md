# Introduction

Vulnhub 是一个提供各种漏洞环境的靶场平台，大部分环境是做好的虚拟机镜像文件，镜像预先设计了多种漏洞，需要使用 VMware 或者 VirtualBox 运行。

每个镜像会有破解的目标，挑战的目标是获取操作系统的 root 权限和查看 flag。

这个集合记录了一些 Vulnhub 上的漏洞环境的复现过程，以及一些漏洞的分析思路。
